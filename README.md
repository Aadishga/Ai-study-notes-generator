# Ai-study-notes-generator
AI-powered study notes generator built with Python and Google Gemini API. Takes any topic and difficulty level as input and automatically generates structured study notes including summary, key concepts, definitions, real world examples, and a mini quiz. Saves output as a text file automatically. Tech Stack: Python, Google Gemini API, Prompt Eng.
## How to Run
1. Clone this repo
2. Install dependencies: pip install google-generativeai
3. Add your Gemini API key (get free key at aistudio.google.com)
4. Run the notebook

## Features
- Generates structured study notes on any topic
- Adjustable difficulty: beginner / intermediate / advanced
- Auto-saves notes as a text file
- Sections: Summary, Key Concepts, Definitions, 
  Real World Example, Mini Quiz, Key Takeaway

## Tech Stack
- Python
- Google Gemini API
- Prompt Engineering
